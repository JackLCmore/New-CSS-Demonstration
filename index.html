Css Snippet cheatsheet

This shows CSS snippets, and it looks good

Flexbox Row

Use those three properties to create a flexbox row layout


            .row{
                display: flex;
                flex-direction: row;
                flex-wrap:wrap;
              }

Flexbox Column

Use this to create a flexbox column layout


                .column{
                    display: flex;
                    flex-direction: column;
                    flex-wrap:wrap;
                  }

CSS Grid Layout

Build a twelve column layout using css grid

                
            .grid{
                display: grid;
                width:100%;
                grid-template-columns:
                repeat (12, 1fr)
              }

Linear Gradients

This will create a background linear gradient from top to bottom 

            .linear-gradient-background{
              background-image:linear-
              gradient(
                rgba(232,102,236,0.3)0%.
                rgna(232,102,236,0,6)100%
              );
              }

Box Transition Glow

Use transition and box shadows to glow on hover

          .code-card .card-header{
              background-image:linear-
              gradient(
                rgba(232,102,236,0.3)0%.
                rgna(232,102,236,0,6)100%
              );
              }

Overlay Card w. Title

Use position properties in negative margin to raise the element higher than their natural starting 

            .card-header{
              position:relative;
              margin-top:-20px;
            }
